## System Requirements

<Info>
Revize is compatible with all Shopify plans and most themes. No coding required.
</Info>

### Shopify Compatibility
- **Shopify Plans**: Works with Basic, Shopify, and Advanced plans
- **Shopify Plus**: Full compatibility with enterprise features
- **Theme Support**: Compatible with all standard Shopify themes
- **Custom Themes**: May require minor adjustments for optimal display

### Required Permissions
When installing Revize, you'll grant the following permissions:
- **Orders**: Read and modify order information
- **Products**: Access product data for editing options
- **Customers**: Access customer information for personalization
- **Store Settings**: Configure app settings within your store

## Installation Process

<Steps>
<Step title="Access Shopify App Store">
1. Log into your Shopify admin panel
2. Navigate to **Apps** in the main menu
3. Click **Visit Shopify App Store**
4. Search for "Revize" or visit our [direct link](https://apps.shopify.com/revize)
</Step>

<Step title="Install the App">
1. Click **"Add app"** on the Revize listing page
2. Review the app permissions and click **"Install app"**
3. Wait for the installation to complete (usually 30-60 seconds)
4. You'll be redirected to the Revize setup dashboard
</Step>

<Step title="Complete Initial Setup">
The setup wizard will guide you through:
- Basic configuration settings
- Time limit preferences
- Email notification setup
- Integration options

<Tip>You can always modify these settings later from the Revize dashboard.</Tip>
</Step>
</Steps>

## Troubleshooting Installation

<AccordionGroup>
<Accordion title="App Won't Install" icon="exclamation-triangle">
**Common causes:**
- Insufficient permissions (contact store owner)
- Browser cache issues (clear cache and try again)
- Shopify service outage (check Shopify status page)

**Solution:** Try refreshing the page or contact support at [support@revize.app](mailto:support@revize.app)
</Accordion>

<Accordion title="Dashboard Not Loading" icon="refresh">
**Common causes:**
- Slow internet connection
- Browser compatibility issues
- Ad blockers interfering

**Solution:** Try a different browser or disable ad blockers temporarily
</Accordion>

<Accordion title="Missing from Apps List" icon="search">
**Common causes:**
- Installation incomplete
- App was uninstalled accidentally
- Permissions issue

**Solution:** Reinstall the app from the Shopify App Store
</Accordion>
</AccordionGroup>
