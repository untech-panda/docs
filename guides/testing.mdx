## Test your Order Status page

<Steps>
  <Step title="Create a draft order">
    In Shopify admin, go to <strong>Orders → Drafts</strong>, then click <strong>Create order</strong>. Add a test customer and a product. Save the draft.
    <Tip>
      New to draft orders? Watch Shopify’s quick video: <a href="https://www.youtube.com/watch?v=qyCPbXqr_ow" target="_blank">How to create a draft order</a>.
    </Tip>
  </Step>

  <Step title="Convert the draft to an order">
    Click <strong>Collect payment</strong> and choose <strong>Mark as paid</strong> for testing. This creates a real order you can view.
  </Step>

  <Step title="Open the Order Status page">
    From the order, open the overflow menu and select <strong>View order status page</strong>. Confirm the Revize blocks appear and load correctly.
  </Step>

  <Step title="Make a quick edit">
    Use the Revize entry point on the Order Status page to change a small detail, for example quantity or shipping address, then save the edit.
  </Step>
</Steps>

<Note>
If you use a staging product or a 100% discount, remember to switch back to real settings before you go live.
</Note>
