## Install Revize in 3 Easy Steps

Get your order editing system up and running quickly with this streamlined setup process.

<Steps>
<Step title="Install from Shopify App Store">

Visit the [Revize app listing](https://apps.shopify.com/revize) in the Shopify App Store and click **"Add app"**.

<Frame>
  <img src="/images/checks-passed.png" alt="Shopify App Store installation" style={{ borderRadius: '0.5rem' }} />
</Frame>

The app will automatically integrate with your store's order system.

</Step>

<Step title="Configure Basic Settings">

Once installed, you'll be redirected to the Revize dashboard where you can configure:

<AccordionGroup>
  <Accordion title="Set Order Edit Time Limits" icon="clock">
    - Choose how long customers have to edit orders (default: 30 minutes)
    - Set different limits for different customer segments if needed
    - Configure timezone settings for global stores
  </Accordion>
  
  <Accordion title="Configure Refund Options" icon="credit-card">
    - Enable automatic refunds for removed items
    - Set refund processing preferences
    - Configure partial refund handling
  </Accordion>
  
  <Accordion title="Choose Edit Options" icon="edit">
    - Allow address changes
    - Enable product swapping
    - Set quantity change limits
    - Configure upsell opportunities
  </Accordion>
</AccordionGroup>

</Step>

<Step title="Add to Your Store">

Complete the integration by adding Revize to your customer experience:

1. **Email Integration**: Revize automatically adds edit links to order confirmation emails
2. **Thank You Page**: The edit option appears on post-checkout pages
3. **Customer Portal**: Links are available in customer accounts

<Tip>Test the setup using Shopify's draft order feature before going live!</Tip>

</Step>
</Steps>

## What Happens Next?

Once installed, Revize will:

1. **Hold New Orders**: Orders are automatically put on hold for the configured time period
2. **Send Edit Links**: Customers receive editing options via email and account portal
3. **Process Changes**: Customer modifications are applied in real-time
4. **Release Orders**: After the edit window closes, orders proceed to fulfillment
